# lcsaj
## Level 3: Nested Loops

This challenge is named `level-3`, and is located within module `lcsaj`.

When users start this challenge, their environment will have:
- `/flag`
- `/challenge/level-3`

**Goal**: Use `lcsajdump /challenge/level-3` to find the loop LCSAJ (the
back-edge path) and count how many times it executes.  The loop builds a token
by prepending the current string length on each iteration.  Trace the LCSAJ
to derive the final token, then supply it to the program.

```
$ /challenge/level-3
Token: <derived-token>
pwn.college{...}
```
