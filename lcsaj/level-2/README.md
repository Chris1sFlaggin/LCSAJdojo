# lcsaj
## Level 2: Conditional Branches

This challenge is named `level-2`, and is located within module `lcsaj`.

When users start this challenge, their environment will have:
- `/flag`
- `/challenge/level-2`

**Goal**: Use `lcsajdump /challenge/level-2` to enumerate all LCSAJs.
The program has two independent branch points (Value A and Value B).
Only the LCSAJ path that passes both checks leads to the flag.

```
$ /challenge/level-2
Value A: alpha
Value B: omega
pwn.college{...}
```
