# lcsaj
## Level 1: Sequential Path

This challenge is named `level-1`, and is located within module `lcsaj`.

When users start this challenge, their environment will have:
- `/flag`
- `/challenge/level-1`

**Goal**: Use `lcsajdump /challenge/level-1` to inspect the program's single
Linear Code Sequence and Jump (LCSAJ).  The dump will reveal the one passphrase
that leads straight through to the flag.

Provide that passphrase when prompted:

```
$ /challenge/level-1
Passphrase: lcsaj
pwn.college{...}
```
